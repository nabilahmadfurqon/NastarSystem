# Nastar Sheet.best App

## Setup
1) npm install
2) cp .env.example .env
3) isi VITE_SHEETBEST_BASE_URL
4) npm run dev

## Google Sheets tabs
- PESANAN
- PRODUKSI
- BAHAN_BAKU

## Columns wajib
PESANAN:
ID, Tanggal, Nama, Ukuran, JumlahToples, Total, StatusPesanan, StatusBayar

PRODUKSI:
Ukuran, Gram, ModalPerToples, HargaJualPerToples

BAHAN_BAKU:
Bahan, Harga, Satuan, Stok, Untuk1kgNastar
